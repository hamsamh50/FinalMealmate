 🍽️ MealMate - Django Food Delivery System
 
📌 Overview

MealMate is a full-stack food delivery web application built using Django. It allows users to browse restaurants, view menus, add items to cart, and place orders.

 🚀 Features

* User authentication (Signup/Login)
* Restaurant listing and management
* Menu browsing system
* Cart functionality
* Order placement system
* Admin dashboard for management

 🛠️ Tech Stack

* Python
* Django
* SQLite
* HTML, CSS, Bootstrap
* JavaScript

📂 Project Structure

* delivery/ → Main app (restaurants, orders, cart)
* meal_buddy/ → Django project settings
* static/ → CSS & images
* manage.py → Django entry point

⚙️ Setup Instructions

```bash
git clone https://github.com/hamsamh50/FinalMealmate.git
cd FinalMealmate
pip install -r requirements.txt
python manage.py runserver
```
